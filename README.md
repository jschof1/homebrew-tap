# Homebrew Tap for youtune

Install [youtune](https://github.com/jschof1/youtune) via Homebrew.

## Install

```bash
brew tap jschof1/tap
brew install youtune
```

## What is youtune?

The smartest YouTube → MP3 downloader. Auto-tags with MusicBrainz, embeds album art, fetches lyrics, upgrades quality via Soulseek.

```bash
youtune "https://youtube.com/watch?v=dQw4w9WgXcQ"
```
